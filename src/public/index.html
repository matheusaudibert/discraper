How?

Our site allows you to collect information from Discord servers by providing your account token and the server ID. Using a selfbot, the tool scans all members of the specified server and captures their IDs. It then performs detailed scraping, focusing only on members who created their accounts before 2018. Your token is used exclusively during the process and is not stored, ensuring your data remains secure. To use the tool, you must be a member of the target server. The entire process is efficient, reliable, and designed with your privacy in mind.

API

The application uses the API (discordlookup.mesalytic.moe/v1/user/:id) to fetch user information efficiently. This API allows us to retrieve detailed data about Discord users, including their public profile information and, consequently, their badges. By leveraging this API, we ensure accurate and up-to-date results while maintaining a secure and reliable process for accessing user data.
